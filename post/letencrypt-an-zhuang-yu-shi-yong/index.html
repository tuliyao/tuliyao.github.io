letencrypt安装与使用

2023-02-22 1 min read

1.安装certbot
yum install certbot

2.获取证书
certbot certonly
根据命令操作

3.查看证书
certbot certificates

4.定期更新证书
certbot renew

4.1 设置任务计划定时更新
0 1 1 */2 * /usr/local/cronlist/renewletsencrypt.sh &

sh脚本内容：
#!/bin/bash
sudo certbot renew;
sudo /etc/init.d/nginx reload;
