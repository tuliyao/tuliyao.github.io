1.安装certbot
没安装nginx需安装nginx插件, 端口80要开放，且没有被占用
yum install python-certbot-nginx -y
yum install certbot

2.获取证书
方式一： certbot certonly -d 域名 --nginx
方式二：
certbot certonly
步骤1：初次使用第一步需要输入邮箱接收通知验证。后面再次申请证书时不会再弹出要求输入邮箱
步骤2： 给出了3个选项，选择选项2
步骤3：输入域名
